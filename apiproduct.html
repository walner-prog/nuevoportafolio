Documentación API de Productos

Documentación API de Productos Esta documentación describe la API de productos implementada en Node.js con Express. La API proporciona endpoints para gestionar productos electrónicos, ropa, libros, juegos de mesa y artículos deportivos. Se simula una base de datos en memoria con productos predefinidos.

Obtener todos los productos
GET /products

Descripción:

Devuelve todos los productos disponibles en la tienda.

Respuesta Exitosa:

Código de estado: 200 OK Cuerpo de respuesta: Array de objetos de productos.

Respuesta de Error: Código de estado: 404 Not Found Cuerpo de respuesta: { "message": "No hay productos disponibles" }

2. Obtener un producto por ID

- GET /products/:id
-Parámetros de Ruta:

id: Identificador único del producto.

Descripción:

Devuelve un producto específico según el ID proporcionado.

Respuesta Exitosa:

Código de estado: 200 OK Cuerpo de respuesta: Objeto del producto.

Respuesta de error:

Código de estado: 404 Not Found Cuerpo de respuesta: { "message": "Producto no encontrado" }

3. Obtener productos por categoría

! GET /products/category/:category

Parámetros de Ruta:

category:

Nombre de la categoría.

Descripción:
Devuelve todos los productos de una categoría específica.

Respuesta Exitosa:

Código de estado: 200 OK Cuerpo de respuesta: Array de objetos de productos. Respuesta de Error: Código de estado: 404 Not Found Cuerpo de respuesta: { "message": "No hay productos en esta categoría" }

4. Obtener un producto por ID dentro de una categoría

GET /products/category/:category/:id

Parámetros de Ruta:

category: Nombre de la categoría.

id: Identificador único del producto.

Descripción: Devuelve un producto específico según el ID y la categoría proporcionados.

Respuesta Exitosa: Código de estado: 200 OK Cuerpo de respuesta: Objeto del producto.

Respuesta de Error: Código de estado: 404 Not Found Cuerpo de respuesta: { "message": "Producto no encontrado en esta categoría" }
